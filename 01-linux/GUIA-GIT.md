# Guia de Comandos Git Essenciais

1. **git init**
   * *O que faz:* Inicia um repositório novo na pasta.
   * *Quando usar:* Apenas na criação do projeto.

2. **git status**
   * *O que faz:* Mostra o estado dos arquivos (modificados, novos, etc).
   * *Quando usar:* O tempo todo, para não se perder.

3. **git add .**
   * *O que faz:* Prepara todos os arquivos modificados para o próximo commit.
   * *Quando usar:* Quando terminou uma etapa do trabalho.
